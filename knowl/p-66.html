Suppose that \(I\) is a nonzero prime ideal. Let \(g,h\in F[X]\) be such that \(f=gh\text{.}\) Hence \(gh\in I\text{.}\) Since \(I\) is a prime ideal, either \(g\in I\) or \(h\in I\text{.}\) If \(g\in I\) then, using ObservationÂ 4.5.2, we get that \(g=uf\) for some unit \(u\text{.}\) Similar arguments shows that if \(h\in I\) then \(h=u^\prime f\) for some unit \(u^\prime\text{.}\) Hence, \(f\) is irreducible.
