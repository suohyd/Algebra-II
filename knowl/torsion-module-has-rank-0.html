Example 9.7.6. (Rank of a torsion module is zero).

Let \(M\) be a left module over an integral domain \(A\text{,}\) and assume that \(M={\rm Tor}(M)\text{.}\) For any \(m\in M\) there exists \(0\neq a\in A\) such that \(a\cdot m=0\text{,}\) i.e., every \(\{m\}\) is \(A\)-linearly dependent. Thus the rank of a torsion module over an integral domain is \(0\text{.}\)
